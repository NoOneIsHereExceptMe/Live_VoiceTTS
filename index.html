Audio Routing Setup

To route the synthesized voice to both your speakers and other applications:

  1. Install a virtual audio cable software like VB-Audio Virtual Cable or Voicemeeter.
  2. Set your virtual cable as the default audio output device in your system settings.
  3. In Voicemeeter (or similar software), configure it to forward audio to both your physical speakers and the virtual output.
  4. Use the virtual output as an input in your target applications.

This allows you to use the converted voice in other applications like streaming software, voice chat, or recording tools.
